Text Formatting using HTML

  • Bold Text - Enclose text in "b" attribute
  • Strong Text
    - Difference can be seen while using Text-to-Speech.
    - Enclose text in "strong" attribute
  • Italic Text - Enclose text in "i" attribute
  • Emphasis Text
    - Difference can be seen while using Text-to-Speech.
    - Enclose text in "em" attribute
  • UnderLined Text - Enclose Text in "u" attribute

  • Struck Text - Enclose Text either in "s" or "strike" or "del" attributes

  • Subscript Text Main Text Superscript Text - Enclose in "sub" or "sup" attributes as required.

  • Small Text - Enclose text in "small" attribute

  • Big Text - Enclose text in "big" attribute

  • Marked Text - Enclose text in "mark" attribute

  • Hover mouse over Me! - Enclose text in "abbr" attribute and to provide on Hover text, use "abbr title".
  • Inserted Text
    - Difference can be seen while using Text-to-Speech.
    - Enclose text in "ins" attribute

    Table in HTML

    • Start with Table attribute
    • Define a ROW for table as heading, tr > th
    • tr stands for Table ROW
    • th stands for Table HEAD
    • Close table row after Head Row completed.
    • Start new ROW
    • This time use attribute td for row's data
    • td stands for Table DATA
    • Add as many required rows, style it if required.
    • Close Table attribute.
